# Fourier Transform & RLC System Analysis  
ELI220 – Practical Report 2  

## 📌 Overview

This project investigates the frequency-domain analysis of pulse trains and linear time-invariant (LTI) systems using Fourier and Laplace transforms.

Theoretical derivations were validated through circuit simulations using LTspice and graphical analysis in Python.

---

## 🎯 Objectives

- Derive Fourier transforms of pulse trains
- Analyse frequency-domain representations of signals
- Derive transfer functions H(s) and H(f) for RC, RL, and RLC circuits
- Study overdamped, underdamped, and critically damped behaviour
- Compare theoretical results with LTspice simulations
- Plot magnitude responses |H(f)| and impulse responses h(t)

---

## 🛠 Tools & Technologies Used

- Python  
- NumPy  
- Matplotlib  
- LTspice  
- Fourier Transform theory  
- Laplace Transform theory  
- Linear Time-Invariant (LTI) system modelling

## 🔬 Systems Analysed

### Signal Analysis
- Pulse train approximation using Fourier Transform
- Sinc function behaviour
- Frequency domain impulse sampling

### Circuit Analysis
- RC Circuit
- RL Circuit
- RLC Circuit (multiple damping configurations)
    - Overdamped
    - Underdamped
    - Critically damped

Transfer functions were derived analytically and compared to simulated results.
